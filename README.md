# Akilli
Akıllı kapan-Bu kodları Arduino'ya atarak "Akıllı Kapan" çalışmamızı yapabilirsiniz.

Akıllı Kapan Ne İşe Yarar?

Tasarlamış olduğumuz akıllı kapan ile her türlü alandan kaçak geçişleri önlememiz mümkün. Örnek olarak; sınır kapısından Türkiye Cumhuriyeti’ne ürün getiren birisi geçmek istiyor. Eğer sınırda durup güvenlik güçlerinden izin almazsa sınır kapısının geçişinde bulunan gizli kapan ani olarak açılarak kaçak geçişi önleyecek. Eğer sınır kapısında durup güvenlik güçlerinden gerekli izinlerini alırsa güvenlik güçleri güvenlik odasında bulunan buton ile kapıdaki sensörümüzü pasif hale getirecek ve araç geçerken kapan açılmayacak. Daha sonra tekrardan butona basılacak ve sensörümüz aktif hale getirilerek güvenlik sağlanmaya devam edilecektir.

Akıllı Kapan’ın Yazılımı Nasıl Atıldı?

Arduino DUE içerisindeki Atmel SAM3X8E ARM Cortex-M3 CPU sayesinde kolayca yazılım yükleyerek fiziksel hayatı kontrol edebiliyoruz. Öncelikle internet üzerinden kurduğumuz Arduino IDE 1.8.7 programı ile birlikte Arduino DUE’nin gerekli yazılımını yazıyoruz. Eğer Arduino DUE Arduino IDE programımızda yüklü değilse Kart Yöneticisi yardımı ile Arduino DUE’yi kuruyoruz ve gerekli yazılımı yüklüyoruz.
